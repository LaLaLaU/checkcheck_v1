# CheckCheck 导管喷码自动核对系统

CheckCheck是一个基于计算机视觉和OCR技术的导管喷码自动核对系统，旨在提高导管喷码与金属标牌文字核对的效率和准确性。

## 项目概述

系统通过工业相机或上传图片的方式捕获金属标牌和管身喷码，利用OCR技术自动识别并比对两处文字，实现高效、准确的核对工作。

## 功能特点

- 支持图片上传和识别
- 自动检测标牌和喷码区域
- 高精度OCR文字识别
- 智能文本比对
- 结果记录和查询

## 安装与使用

### 环境要求

- Python 3.8+
- Windows 10/11操作系统

### 安装步骤

1. 克隆或下载项目代码
2. 安装依赖库：
   ```
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```
   python src/main.py
   ```
